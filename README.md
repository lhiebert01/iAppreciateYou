### Quick Start
Simply open `index.html` in your web browser - that's it! No installation, no dependencies, no complicated setup.

```bash
# Clone the repository
git clone https://github.com/lhiebert01/iAppreciateYou.git

# Navigate to the directory

# Open in your browser
```
