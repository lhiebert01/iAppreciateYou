## Features

### Core Functionality
- **400+ Super-Affirmations** - Carefully crafted affirmations across 8 categories
- **Rich Learning Experience** - Each affirmation includes meaning and reinforcement details
- **Favorites System** - Save and search your favorite affirmations
- **Progress Tracking** - Daily count, favorites count, and streak tracking
- **Confetti Celebrations** - Delightful visual effects with each affirmation
- **Share Feature** - Easy sharing via Web Share API or clipboard
- **Fully Responsive** - Beautiful on mobile, tablet, and desktop
- **Offline Capable** - All data stored locally in your browser

### 8 Affirmation Categories

| Category | Focus |
|----------|-------|
| **Winner** | Championship mindset, victory, excellence |
| **Confidence** | Self-belief, courage, boldness |
| **Gratitude** | Appreciation, thankfulness, abundance |
| **Strength** | Resilience, power, perseverance |
| **Self-Love** | Self-care, acceptance, compassion |
| **Success** | Achievement, goals, prosperity |
| **Mindset** | Growth, positivity, mental wellness |

## Getting Started

### Quick Start

Simply open `index.html` in your browser - no installation required!

```bash
# Clone the repository
git clone https://github.com/lhiebert01/iAppreciateYou.git

# Navigate to the directory
cd iAppreciateYou

# Open in your browser
open index.html  # macOS
start index.html # Windows
```

### Using a Local Server (Optional)

```bash
# Using Python 3
python3 -m http.server 8080

# Then visit http://localhost:8080
```

### Deploy to GitHub Pages

1. Push to your GitHub repository
2. Go to Settings > Pages
3. Select "Deploy from a branch" and choose `main`
4. Your app will be live at `https://yourusername.github.io/iAppreciateYou/`

## How to Use

1. **Get Your First Super-Affirmation** - Click "Super-Affirmation" to receive a powerful message
2. **Learn More** - Click the "Learn More" button to see the meaning and reinforcement
3. **Explore Categories** - Use the category buttons to filter by theme
4. **Save Favorites** - Click the heart icon to save affirmations you love
5. **Search Favorites** - Use the search bar to find saved affirmations
6. **Track Progress** - Watch your daily count and streak grow
7. **Share the Love** - Use the Share button to spread positivity
8. **About** - Click the "i" button to learn more about the app
